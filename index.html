FAQ

Common Questions

Does ClutchReframe Live provide any competitive advantage?

No. ClutchReframe Live is a stream production tool that only affects what your viewers see. Your own in-game view is never changed — no overlays, no extra information, no advantage. Camera decisions are based on visual analysis of the broadcast feed, not game memory. It does not track abilities, cooldowns, or any competitive game state.

What data does the plugin process?

ClutchReframe Live analyzes your screen capture locally and in real-time to detect game events and drive camera movements. Frames are processed on-the-fly and never stored or uploaded. The plugin does not read competitive game state. It also stores your local configuration preferences. No personal information is collected or transmitted.

Does the plugin read game memory or modify game files?

No. ClutchReframe Live works entirely through screen capture and OBS Studio's plugin interface. It does not access game memory, modify game files, or inject code into the game process.

Will it affect my game performance?

ClutchReframe Live is designed to be lightweight. The analysis runs as a separate background process, keeping impact on game performance minimal. The OBS plugin itself adds negligible overhead to your stream encoding pipeline.

Is ClutchReframe Live affiliated with Riot Games or OBS?

No. ClutchReframe Live is an independent plugin. It is not endorsed by or affiliated with Riot Games or the OBS Project in any way.

Is the plugin open-source?

The OBS filter plugin is open-source (GPLv2) as required by the OBS Studio license, covering only the video filter layer that receives ROI coordinates and applies the crop. The visual recognition engine is a completely separate, closed-source local application.

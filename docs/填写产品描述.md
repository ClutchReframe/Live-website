# ClutchReframe Live

完整版英文：

ClutchReframe Live is a real-time automatic camera work plugin for League of Legends live streams, built as an OBS Studio video filter.

Core Features:
1. AI-Powered Camera Work — Continuously directs the broadcast camera throughout the entire game, just like a professional esports camera operator. Intelligently crops and reframes the game view to fit multiple output aspect ratios (including 9:16 vertical and 16:9 horizontal), smoothly panning and zooming to follow laning, rotations, teamfights, and objectives in real time. Viewers experience polished, broadcast-quality camera work optimized for their viewing platform while the streamer's own game view stays completely unchanged.
2. OBS Video Filter Architecture — Operates as a standard OBS Studio video filter applied to a video source, working entirely within the OBS video processing pipeline. No in-game overlay, no screen injection, no interaction with the game client. The architecture consists of two independent components: a separate local application performs visual analysis of the screen capture and outputs ROI (Region of Interest) coordinates — the target region the camera should focus on. The OBS filter plugin receives these coordinates and applies the video crop.
3. Fully Local Processing — Every camera decision, every frame analysis, every setting, processed entirely on the user's machine. No cloud servers. No data uploads. Your stream stays yours.

This application does not use any Riot Games API endpoints. It does not access game memory, modify game files, or inject code into the game process. All camera decisions are based purely on visual analysis of the game monitor's screen capture. The plugin does not provide any competitive advantage. It does not extract or track specific competitive data such as abilities, cooldowns, health bars, minimap positions, or any other individual game metrics. The streamer's in-game view is never altered; the plugin only affects what viewers see on the broadcast.

We are registering to comply with Riot's third-party application policies.

中文翻译：

ClutchReframe Live 是一款用于英雄联盟直播的实时自动运镜插件，作为 OBS Studio 视频滤镜构建。

核心功能：
1. AI 驱动的智能运镜 — 如同职业电竞赛事摄像师一样，在整场比赛中持续控制直播画面。支持多种输出比例（如 9:16 竖屏、16:9 横屏），智能裁剪并重构游戏画面，实时平滑地平移和缩放，跟随对线、转线、团战和目标争夺。观众在任意平台上获得专业的赛事级运镜体验，而主播自己的游戏画面完全不受影响。
2. OBS 视频滤镜架构 — 作为标准的 OBS Studio 视频滤镜应用于视频源，完全在 OBS 视频处理管线中工作。无游戏内叠加层、无屏幕注入、与游戏客户端无任何交互。架构由两个独立组件组成：独立的本地应用程序对屏幕画面进行视觉分析，输出 ROI（Region of Interest，感兴趣区域）坐标，即镜头应聚焦的目标区域。OBS 滤镜插件接收这些坐标并执行视频裁剪。
3. 完全本地处理 — 每个运镜决策、每帧画面分析、每项设置，全部在用户本机处理。无云服务器，无数据上传。你的直播属于你自己。

本应用不使用任何 Riot Games API 端点。不访问游戏内存、不修改游戏文件、不向游戏进程注入代码。所有运镜决策完全基于对游戏所在显示器画面的视觉分析。插件不提供任何竞争优势，不提取或追踪技能、冷却时间、血条、小地图位置或任何具体的竞技数据指标。主播的游戏内画面从不被改变，插件仅影响观众在直播中看到的画面。

我们注册是为了遵守 Riot 第三方应用政策。
